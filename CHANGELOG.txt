Douban 1.x Changelog
---------------------

1.x to 1.1 20110224
----------
- Add Douban account binding feature, a tab shows on the user page(/user/%uid) which make users are able to bind or unbind douban account.
- Add Douban user list(admin/settings/douban/users), administrators are able to see who using douban account to login. and also able to do some handle, such as unbind or delete account.
